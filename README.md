## 프로젝트 설치

### clone repository

```
git init
git clone https://github.com/dosion9/nbc-react-ts-todo
```

### Install npm dependencies

```
npm install
```

### Start dev-server

```
npm run start
```

### Start json-server

```
json-server --watch db.json --port 4000
```
